# 执行远程部署说明

由于当前 PowerShell 环境中没有 bash 命令，请使用以下方式之一执行远程部署：

---

## 🚀 方案 1: 使用 Git Bash（推荐）

### 步骤：

1. **打开 Git Bash**
   - 在项目目录右键 → `Git Bash Here`
   - 或从开始菜单打开 `Git Bash`

2. **执行部署命令**：
   ```bash
   # 设置环境变量
   export DEPLOY_HOST=165.154.233.55
   export DEPLOY_USER=ubuntu
   export DEPLOY_PATH=/opt/redpacket
   export DEPLOY_BRANCH=master
   
   # 执行部署脚本
   bash deploy/scripts/deploy_remote.sh
   ```

---

## 🐧 方案 2: 使用 WSL（如果已安装）

### 步骤：

1. **打开 WSL**：
   ```powershell
   wsl
   ```

2. **进入项目目录**（需要先在 WSL 中访问 Windows 文件系统）：
   ```bash
   cd /mnt/e/002-工作文件/重要程序/红包系统机器人/037重新开发新功能
   ```

3. **执行部署命令**：
   ```bash
   export DEPLOY_HOST=165.154.233.55
   export DEPLOY_USER=ubuntu
   export DEPLOY_PATH=/opt/redpacket
   export DEPLOY_BRANCH=master
   bash deploy/scripts/deploy_remote.sh
   ```

---

## 🔧 方案 3: 直接在服务器上执行

### 步骤：

1. **SSH 连接到服务器**：
   ```bash
   ssh ubuntu@165.154.233.55
   ```

2. **进入项目目录**：
   ```bash
   cd /opt/redpacket
   ```

3. **执行服务器端部署脚本**：
   ```bash
   # 如果脚本已存在
   bash deploy/scripts/deploy.sh master
   
   # 或使用已有的部署脚本
   bash deploy/scripts/deploy_on_server.sh
   ```

---

## 📝 方案 4: 使用 PowerShell 直接 SSH

如果只需要触发远程脚本，可以直接使用 PowerShell：

```powershell
# 设置变量
$server = "ubuntu@165.154.233.55"
$projectDir = "/opt/redpacket"

# 直接执行远程命令
ssh $server "cd $projectDir && bash deploy/scripts/deploy.sh master"
```

---

## ✅ 推荐流程

**最简单的方式**：

1. 打开 **Git Bash**
2. 执行以下命令（一行一行执行，或一次性复制）：
   ```bash
   export DEPLOY_HOST=165.154.233.55
   export DEPLOY_USER=ubuntu
   export DEPLOY_PATH=/opt/redpacket
   export DEPLOY_BRANCH=master
   bash deploy/scripts/deploy_remote.sh
   ```

---

## 🔍 部署脚本说明

执行 `deploy_remote.sh` 后会自动：

1. ✅ **测试 SSH 连接** - 验证可以连接到服务器
2. ✅ **检查远程脚本** - 如果 `deploy.sh` 不存在会自动上传
3. ✅ **执行远程部署** - 在服务器上运行完整的部署流程：
   - 备份当前版本
   - 拉取最新代码
   - 检查环境变量
   - 停止现有服务
   - 构建 Docker 镜像
   - 启动所有服务
   - 健康检查
   - 显示服务状态

## 📞 需要帮助？

如果遇到问题，请检查：

1. **SSH 连接**：
   ```bash
   ssh ubuntu@165.154.233.55 "echo '连接成功'"
   ```

2. **服务器上的项目目录**：
   ```bash
   ssh ubuntu@165.154.233.55 "ls -la /opt/redpacket"
   ```

3. **部署日志**（部署后查看）：
   ```bash
   ssh ubuntu@165.154.233.55 "tail -f /opt/redpacket/logs/deploy.log"
   ```
