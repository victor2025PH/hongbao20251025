# 🔐 填写防火墙规则 - 详细步骤

## 📋 当前状态

您已经打开了"创建防火墙"页面，防火墙名称已填写为：`redpacket-api-firewall`

---

## 🎯 填写第一条规则：Web Admin API (8000 端口)

### 在当前第一行的规则中填写：

#### 1. 协议列（Protocol）
- 保持第二个下拉框选择：**"指定端口TCP"**
- 第一个下拉框保持默认即可

#### 2. 来源IP列（Source IP）
- ✅ 已经填写：`0.0.0.0/0` （保持不变即可）

#### 3. 端口列（Port）- 重要！⭐
点击端口输入框，填写：**8000**
> 💡 **注意**：直接填写数字 `8000`，不需要任何引号或符号

#### 4. 动作列（Action）
- ✅ 已经选择：**"接受"** （保持不变即可）

#### 5. 优先级列（Priority）
- ✅ 已经选择：**"高"** （保持不变即可）

#### 6. 备注列（Remarks）
点击备注输入框，填写：
```
Web Admin API
```

---

## ➕ 添加第二条规则：MiniApp API (8080 端口)

### 步骤 1: 点击"+"按钮

在第一行规则下方，找到并点击大的 **"+"** 按钮（应该显示在表格中间）。

点击后会添加一行新的规则行。

### 步骤 2: 填写第二条规则

在新添加的第二行规则中，填写：

#### 1. 协议列（Protocol）
- 在第二个下拉框中选择：**"指定端口TCP"**

#### 2. 来源IP列（Source IP）
填写：**0.0.0.0/0**
> 💡 **注意**：直接填写 `0.0.0.0/0`，不需要任何引号或符号

#### 3. 端口列（Port）
填写：**8080**
> 💡 **注意**：直接填写数字 `8080`，不需要任何引号或符号

#### 4. 动作列（Action）
选择：**"接受"**

#### 5. 优先级列（Priority）
选择：**"高"**

#### 6. 备注列（Remarks）
填写：
```
MiniApp API
```

---

## 💾 保存防火墙规则

### 填写完成后：

1. **检查规则**：
   - 确认有 2 条规则
   - 第一条：端口 8000，备注 "Web Admin API"
   - 第二条：端口 8080，备注 "MiniApp API"

2. **查找保存按钮**：
   - 在页面底部或右上角找到 **"确定"**、**"保存"** 或 **"创建"** 按钮
   - 点击保存按钮

3. **应用防火墙组**：
   - 保存后，系统可能会提示是否应用到云主机
   - 如果提示，选择 **"应用到云主机"** 或 **"绑定"**
   - 选择 **UHost-Sale** 云主机
   - 点击 **"确定"**

4. **等待生效**：
   - 等待 1-2 分钟让配置生效
   - 配置完成后，刷新浏览器页面

|------|------|--------|------|------|--------|------|

## ⚠️ 注意事项

1. **端口格式**：直接填写数字即可（如 `8000`），不需要引号或其他符号
2. **协议选择**：确保选择 **"指定端口TCP"**，不要选择其他协议
3. **来源IP**：`0.0.0.0/0` 表示允许所有IP访问（如果只需要特定IP，可以修改）
4. **优先级**：选择"高"可以确保规则优先级较高

## 🆘 如果找不到"+"按钮

如果看不到"+"按钮添加第二条规则，可以：
1. 先保存第一条规则
2. 然后在防火墙组详情页中找到"编辑规则"或"添加规则"按钮
3. 添加第二条规则
