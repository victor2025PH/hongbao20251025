## 📝 测试脚本使用说明

### test-api.ps1

基础测试脚本，快速验证 API 是否可访问。

**使用方法**:
```powershell
.\test-api.ps1
```

**输出**: 
- 每个 API 的访问状态
- HTTP 状态码
- 响应内容

---

### run-comprehensive-test.ps1

全面测试脚本，执行多个端点的测试。

**使用方法**:
```powershell
.\run-comprehensive-test.ps1
```

**输出**:
- 所有测试端点的结果
- 测试汇总统计
- CSV 格式的测试结果文件
