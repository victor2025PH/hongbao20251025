## 🚀 启动服务

### 方式 1: 使用手动启动脚本（推荐）

```powershell
$env:PYTHONIOENCODING="utf-8"
$env:PYTHONUTF8="1"
python start_services_manual.py
```

这个脚本会：
1. 初始化数据库
2. 启动 Web Admin (端口 8001)
3. 启动 MiniApp API (端口 8080)
4. 等待服务就绪
5. 执行健康检查
6. 持续运行直到按 Ctrl+C

### 方式 2: 分别启动（用于调试）

**终端 1 - Web Admin**:
```powershell
$env:PYTHONIOENCODING="utf-8"
python -m uvicorn web_admin.main:app --host 0.0.0.0 --port 8001 --reload
```

**终端 2 - MiniApp API**:
```powershell
$env:PYTHONIOENCODING="utf-8"
python -m uvicorn miniapp.main:app --host 0.0.0.0 --port 8080 --reload
```

**终端 3 - 测试**:
```powershell
$env:PYTHONIOENCODING="utf-8"
python test_services.py
```

### 方式 3: 使用批处理文件

直接双击运行 `快速启动.bat`

## 🔧 故障排查

### 问题 1: 服务无法启动

**检查步骤**:
1. 检查端口是否被占用:
   ```powershell
   netstat -ano | Select-String -Pattern ":(8001|8080)"
   ```

2. 检查模块导入:
   ```powershell
   python -c "from web_admin.main import app; print('OK')"
   python -c "from miniapp.main import app; print('OK')"
   ```

3. 查看错误日志:
   - `web_admin.log`
   - `miniapp.log`

### 问题 2: 数据库错误

**解决方案**:
```powershell
python -c "from models.db import init_db; init_db()"
```

### 问题 3: 编码问题

**解决方案**:
```powershell
# 设置环境变量
$env:PYTHONIOENCODING="utf-8"
$env:PYTHONUTF8="1"

# 或运行编码修复脚本
powershell -ExecutionPolicy Bypass -File fix_encoding.ps1
```
