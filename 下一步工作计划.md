## 📝 注意事项

### 测试编写规范
1. **使用统一的测试基础设施** - 利用 `conftest.py` 中的 fixtures
2. **Mock 外部依赖** - 避免实际网络请求和数据库操作
3. **测试隔离** - 每个测试应该独立，不依赖其他测试
4. **清晰的测试名称** - 使用描述性的测试函数名

### Bot 测试特殊考虑
1. **使用 aiogram 测试工具** - `Dispatcher` 和 `Bot` 的 mock
2. **测试异步函数** - 使用 `pytest-asyncio`
3. **模拟 Telegram API** - 避免实际调用 Telegram API
