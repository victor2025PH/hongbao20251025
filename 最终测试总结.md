## 🚀 启动服务

### 推荐方式: 使用手动启动脚本

```powershell
$env:PYTHONIOENCODING="utf-8"
$env:PYTHONUTF8="1"
python start_services_manual.py
```

这个脚本会：
1. ✅ 自动初始化数据库
2. ✅ 启动 Web Admin (端口 8001)
3. ✅ 启动 MiniApp API (端口 8080)
4. ✅ 等待服务就绪
5. ✅ 执行健康检查
6. ✅ 持续运行直到按 Ctrl+C

### 或者分别启动（用于调试）

**终端 1 - Web Admin**:
```powershell
$env:PYTHONIOENCODING="utf-8"
python -m uvicorn web_admin.main:app --host 0.0.0.0 --port 8001 --reload
```

**终端 2 - MiniApp API**:
```powershell
$env:PYTHONIOENCODING="utf-8"
python -m uvicorn miniapp.main:app --host 0.0.0.0 --port 8080 --reload
```

## 📋 服务端点

- Prometheus 指标: http://localhost:8001/metrics

### MiniApp API (端口 8080)
- 健康检查: http://localhost:8080/healthz
- 公开群组列表: http://localhost:8080/v1/groups/public
- 认证登录: http://localhost:8080/api/auth/login
