## 🔍 监控命令

### 查看服务状态
```powershell
python test_services.py
```

### 查看服务日志
```powershell
# Web Admin 日志
Get-Content web_admin.log -Tail 50 -Wait

# MiniApp API 日志
Get-Content miniapp.log -Tail 50 -Wait
```

### 查看进程
```powershell
Get-Process python | Select-Object Id, ProcessName, StartTime
```

### 检查端口
```powershell
netstat -ano | Select-String -Pattern ":(8001|8080)"
```

## 📝 测试报告

测试完成后会生成报告文件：
- 文件名格式: `test_report_YYYYMMDD_HHMMSS.json`
- 包含内容: 测试结果、服务状态、错误记录、修复记录

## 🎯 服务端点

测试成功后可以访问：

- **Web Admin**: http://localhost:8001
  - 健康检查: http://localhost:8001/healthz
  - Dashboard: http://localhost:8001/admin/dashboard
  - 指标: http://localhost:8001/metrics

- **MiniApp API**: http://localhost:8080
  - 健康检查: http://localhost:8080/healthz
  - 公开群组: http://localhost:8080/v1/groups/public

## ⚠️ 注意事项

1. 测试系统会在后台持续运行
2. 按 Ctrl+C 可以停止测试
3. 服务启动需要一些时间（通常 10-30 秒）
4. 如果测试失败，查看日志文件了解详情
