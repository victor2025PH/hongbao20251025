## 技术要点

### FastAPI 依赖注入 Mock
- **正确方式**: 使用 `fastapi_app.dependency_overrides[dep] = mock_function`
- **错误方式**: 使用 `patch("module.dep")`（FastAPI 依赖注入系统不会识别）

### 数据库测试隔离
- 使用独立的 SQLite 数据库文件
- 在测试前清理数据库
- 使用 `db_session.commit()` 确保数据持久化

### 异常处理测试
- 非 API 请求的 401/403 会被重定向（303）
- 400 状态码应该返回 `PlainTextResponse`，不会重定向
- 使用 `follow_redirects=False` 防止自动跟随重定向
