## 🔧 修复的技术细节

### 1. Mock 路径修正
```python
# 错误的做法
with patch("services.ai_service.OpenAI"):  # ❌ 模块级别没有此属性

# 正确的做法
with patch("openai.OpenAI"):  # ✅ 从原始模块 patch
```

### 2. 时区处理
```python
# 错误的做法
now = datetime.utcfromtimestamp(ts)  # ❌ naive datetime

# 正确的做法
now = datetime.fromtimestamp(ts, UTC)  # ✅ aware datetime
```

### 3. 函数参数修正
```python
# 错误的做法
update_report_case(db_session, report_id=1, status="resolved")  # ❌ 缺少 operator_tg_id

# 正确的做法
update_report_case(db_session, report_id=1, operator_tg_id=10001, status="resolved")  # ✅
```

### 4. Settings Mock
```python
# 错误的做法
with patch("services.recharge_service._NP_IPN_SECRET", secret):  # ❌ 忽略了 settings

# 正确的做法
with patch("services.recharge_service.settings") as mock_settings, \
     patch("services.recharge_service._NP_IPN_SECRET", secret):
    mock_settings.NOWPAYMENTS_IPN_SECRET = None  # ✅ 同时 mock settings
```
